# DailyRoutine Learn

Веб-приложение для изучения фронтенд-разработки с материалами по JavaScript, React, TypeScript, Next.js и другим технологиям.

## Возможности

- Чтение структурированных материалов по частям и главам
- Прохождение тестов для проверки знаний
- Практические задания с решениями
- Мобильная оптимизация и PWA поддержка
- Экспорт контента в различных форматах

## Технологии

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 4
- Embla Carousel
- React Markdown

## Быстрый старт

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

## Команды

- `npm run dev` - запуск dev сервера
- `npm run build` - сборка для production
- `npm start` - запуск production сборки
- `npm run lint` - проверка кода
- `npm run format` - форматирование кода
